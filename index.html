Smart Brain

The smart brain application uses the Clarifai face detection API to take in a photo, detect and mark a box around the face. It also keeps a record of users login details, hash encrypted passwords and how many photos they have used the API with. Its front end is built using React, Flexbox, Tacyons and uses Github pages. The Back end is built using PostgreSQL.js, Express.js, bcrypt-node.js and hosted on Heroku.
