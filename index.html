International Engineers Certificate

This is a landing page that I built for a international event organisation. I used a Lamda function to collect interest for a new company product using DynamoDB and a existing send grid account to send a confirmation email that was designed in Adobe Dreamweaver. It's created with react and is a static page served from a AWS S3 bucket using Serverless to develop, deploy, and monitor the page.

Face Fixer 1.0/2.0

1.0 - The Face Fixer application uses the Clarifai face detection API to take in a photo, detect and place a resized CSS emoji with a perfect smile over it. It also keeps a record of users login details, hash encrypted passwords and how many photos they have used the API with.
Its front end is built using React, Flexbox, Tacyons. The Back end is built using PostgreSQL.js, Express.js, bcrypt-node.js and hosted on Heroku.
2.0 - The upgraded version of face fixer has been dockerised, had session management implemented using JWT and Redis database. User profile editing options added aswell.
